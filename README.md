# MindCare - DS MINI PROJECT 🧠✨

**Live Demo:** [https://mindcare-project-nujg.onrender.com](https://mindcare-project-nujg.onrender.com)

MindCare is a modern, fully responsive wellness platform designed for early detection and management of depression and stress among youth. It combines data-driven assessments with a premium, soothing UI to provide immediate support.

## 🚀 Features

- **Science-based Assessment**: A 30-minute tailored evaluation to detect stress and depression levels.
- **Stress History & Trend Analysis**: Persistent tracking of results with automatic "Improving/Stable/Worsening" trend detection.
- **MindCare AI Assistant**: A smart, context-aware chatbot that offers personalized guidance and action suggestions.
- **"Calm Me" Mode**: A one-click immersive experience with breathing guides and ambient audio for instant relaxation.
- **Guided Mini Exercises**: A hub for breathing, meditation, grounding, and relaxation sessions.
- **Smart Recommendations**: Stress-based suggestions for TV shows and music playlists to help users unwind.
- **Premium UX/UI**: Liquid cursor ripple effects, natural wind animations, and a distraction-free glassmorphic design.
- **Multi-language Support**: Full support for both English and Hindi users.

## 🛠️ Built With

- **Frontend**: Vite, React, TypeScript, Framer Motion, Tailwind CSS, Lucide React
- **Backend**: Node.js, Express, Nodemailer
- **State**: localStorage-based persistence for privacy and speed

## 🏗️ Getting Started

1. **Install dependencies**
```bash
npm install && cd server && npm install && cd ..
```

2. **Start Development**
- Frontend: `npm run dev`
- Backend: `cd server && npm run dev`

3. **Build & Start Fullstack**
```bash
npm run build:full
npm start
```
