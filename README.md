# MindCare

A safe platform for early detection of depression among youth. Science-based assessment and mental health resources.

## Built With

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Getting Started

1. Install dependencies
```bash
npm install
```

2. Start the development server
```bash
npm run dev
```

3. Build for production
```bash
npm run build
```
